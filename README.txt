Instructions:
1. Open two separate terminals or tabs

2. Compile both files
    >javac SIPServer.java SIPClient.java

3. Run in this order:
    > java SIPServer
    > java SIPClient


Features:
- SIP HANDSHAKE
- SDP Negotiation
- RTP Packet Generation
- RTP Receiving and Playback
- RTCP Usage
- Call Teardown
- Documentation and Clarity
